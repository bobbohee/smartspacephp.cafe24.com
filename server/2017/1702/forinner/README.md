# FORINNER

국내 외국인을 위한 의료 정보 서비스를 제공하는 사이트입니다.

## SKILL

* :hospital: (영어가 가능한)병원 및 약국 정보 서비스

* :bust_in_silhouette: 픽토그램

* :telephone_receiver: 전화 연결

* :microphone: stt(speech to text), tts(text to speech) 및 번역
